17.4 سیکل گرفتن در برره

می‌دانیم سیکل گرفتن جایگاه ویژه‌ای در میان اهالی برره دارد. نظام، شیرفرهاد و کیوون می‌خواهند مدرک سیکل خود را بگیرند در این راستا باید در امتحان تستی‌ای که کَیانوش برای آنها طراحی کرده شرکت کنند.

از آنجایی که این ۳ نفر خواندن و نوشتن بلد نیستند (!) تصمیم می‌گیرند که بدون خواندن سوالات با الگوی تکرار شونده‌ی خاصی تست‌ها‌ را جواب بدهند. امتحانی که کَیانوش طراحی کرده دارای Nسوال ۳ گزینه‌ای است، همچنین در جدول زیر الگویی که هر فرد طبق آن به سوالات پاسخ می‌دهد نشان داده شده است:

دوره تناوب الگو name اسم
۶ ۳, ۳, ۱, ۱, ۲, ۲, ۳, ۳, ۱, ۱, ۲, ۲, … keyvoon کیوون
۳ ۱, ۲, ۳, ۱, ۲, ۳, ۱, ۲, ۳, ۱, ۲, ۳, … nezam نظام
۴ ۲, ۱, ۲, ۳, ۲, ۱, ۲, ۳, ۲, ۱, ۲, ۳, … shir farhad شیرفرهاد

حال ما به شما تعداد و کلید سوالات را می‌دهیم و شما باید بیشترین نمره‌ای که یک فرد از بین این ۳ نفر در امتحان کسب کرده و اسم افرادی را که بیشترین نمره را کسب کرده‌اند به دست آورید.

ورودی

در خط اول ورودی عدد \(N\) آمده که تعداد سوالات را نشان ‌می‌دهد و در خط بعدی رشته‌ای متشکل از اعداد ۱ تا ۳، به طول \(N\) می‌آید که کلید سوالات رامشخص می‌کند (عدد \(i\)ام رشته گزینه‌ی درست برای سوال \(i\)ام امتحان را مشخص می‌کند).

\[ 1 \leq n \leq 100 \]

خروجی

در خط اول خروجی بالاترین نمره‌ای که در امتحان کسب شده را چاپ کنید.

در خطوط بعدی اسم افرادی را چاپ کنید که بالاترین نمره‌ی امتحان را کسب کرده اند (به ترتیب حروف الفبا).

توضیح نمونه ۲:

  • کیوون به سوالات سوم، ششم و هشتم
  • نظام به سوالات اول، هفتم و نهم
  • و شیرفرهاد به سوالات دوم و هشتم پاسخ درست داده‌اند

پاسخ

راه حل. قبل از دیدن پاسخ، سعی کنید خودتان انجام دهید.

Code 
### First Way
keyvoon_answers <- function(n) {
  numbers <- seq_len(n)
  modulus <- numbers %% 6
  keys <- c()
  
  for (number in numbers) {
    if (modulus[number] %in% c(1, 2)) {
      keys[number] <- 3
    } else if (modulus[number] %in% c(3, 4)) {
      keys[number] <- 1
    } else if (modulus[number] %in% c(5, 0)) {
      keys[number] <- 2
    }
  }
  return(keys)
}

nezam_answers <- function(n) {
  numbers <- seq_len(n)
  modulus <- numbers %% 3
  keys <- c()
  
  for (number in numbers) {
    if (modulus[number] %in% c(1)) {
      keys[number] <- 1
    } else if (modulus[number] %in% c(2)) {
      keys[number] <- 2
    } else if (modulus[number] %in% c(0)) {
      keys[number] <- 3
    }
  }
  return(keys)
}


shir_farhad_answers <- function(n) {
  numbers <- seq_len(n)
  modulus <- numbers %% 4
  keys <- c()
  
  for (number in numbers) {
    if (modulus[number] %in% c(1, 3)) {
      keys[number] <- 2
    } else if (modulus[number] %in% c(2)) {
      keys[number] <- 1
    } else if (modulus[number] %in% c(0)) {
      keys[number] <- 3
    }
  }
  return(keys)
}

### Second Way
keys_maker <- function(n, pattern, period) {
  pattern |> 
    strrep(ceiling(n / period)) |> 
    substr(1, n) |>
    convert_single_string_to_vector("") |> 
    as.numeric()
}
keyvoon_answers <- function(n) keys_maker(n, "331122", 6)
nezam_answers <- function(n) keys_maker(n, "123", 3)
shirfarhad_answers <- function(n) keys_maker(n, "2123", 4)


convert_single_string_to_vector <- function(string, pattern) {
  return(strsplit(string, pattern)[[1]])
}
